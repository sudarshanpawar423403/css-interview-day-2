What is box sizing?

  • Box-sizing property determines how the width and height applied on an element.
  • Default value of box-sizing is content box in which actual width of box is equal to padding,border and width of the box.
  • In case of the border box actual width of the box equal to default width of the box,That means in we increase padding and border then also box width remains same.

    • Which property you will use to use to merge cells vertically in grid?

  • Grid-column-start and grid-column-end property used to merge cell vertically. It also have short form i.e grid-column:start/end. Note:these property used only for child element not a parent element.

    • Which property you will use to use to merge cells horizontally in grid?

  • Grid-row-start and grid-row-end property used to merge cell horizontally. It also have short form i.e grid-row:start/end. Note:these property used only for child element not a parent element.
